## About
This is the backend for the skillup mentor project 03: nextup.

It consists of:
- The api is implemented in nestjs
- The database is mongodb
- The reminder service

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
